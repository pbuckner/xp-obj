Adding Objects to Sim
=====================

Once an object is defined using a OBJ8 file, you'll still need to add it to scenery, aircraft or
use a plugin to add it to the running simulator. Here are details for each method:

* :doc:`plugin/adding` using :py:func:`sdk:xp.createInstance` and :py:func:`xp.instanceSetPosition` from :external+sdk:doc:`index`.
  :doc:`sdk:usage/installation_plugin`.
  
* :doc:`scenery/adding` using World Editor (WED).

* :doc:`aircraft/adding` using Plane Maker.
